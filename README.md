# minimalistUI
based on Arch, inspired by EndeavourOS and archinstall command. A small Arch automatic install script project that installs and targets minimalist, simple, and light UI. This project is still heavily WIP.

command to use: 
chmod +x minimalistui.sh
sh minimalistui.sh
